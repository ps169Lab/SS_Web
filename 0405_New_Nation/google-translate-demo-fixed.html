Google Translate Demo (Fixed)

This page demonstrates the Google Translate widget working correctly with the "Powered by Google Translate" text hidden.

CSS to Hide "Powered by Google Translate" Text

The following CSS rule has been added to hide the "Powered by Google Translate" text:

.skiptranslate span {
    display: none !important;
}

This CSS rule targets the span element within elements with the class "skiptranslate", which is the class used by the Google Translate widget.

Implementation in Your Project

I've added this CSS rule to both:

  • 0405_New_Nation/styles.css
  • Web_Template/styles.css

This will hide the "Powered by Google Translate" text in all pages that use these stylesheets.
